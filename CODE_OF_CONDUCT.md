# Code of Conduct 🤝

This repository is intended to be merged back into the original Idleon API Downloader project.

## Expected Behavior ✅

Be respectful, constructive, and specific. Keep discussion focused on improving the extension, documenting issues clearly, and helping users understand limitations.

## Unacceptable Behavior 🚫

Harassment, personal attacks, discriminatory language, spam, or repeated off-topic comments are not welcome in this project.

## Enforcement ⚖️

Maintainers may edit, hide, or remove comments and contributions that do not meet these expectations. Maintainers may also block users from participating if behavior is disruptive or abusive.

## Reporting 📣

Report conduct concerns through the project's GitHub issue or discussion channels if appropriate.
