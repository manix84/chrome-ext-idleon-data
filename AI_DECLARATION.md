# AI Declaration 🤖

This project code is written, reviewed, and maintained by human developers.

AI tools may be used to speed up supporting project work, including documentation drafts, code review notes, commit message suggestions, changelog or release wording, issue triage, and other project paperwork.

AI tools are not treated as project authors, maintainers, or owners. The important design decisions, implementation choices, and code changes are made by human contributors.

In short: the fun and important coding work belongs to people; AI helps reduce the paperwork so contributors can spend more time on the project itself.
