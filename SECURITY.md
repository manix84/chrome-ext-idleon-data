# Security Policy 🛡️

This project is owned and maintained as `Corbeno/Idleon-Api-Downloader`.

## Supported Versions ✅

Only the latest code and latest GitHub Release are considered supported.

## Reporting a Vulnerability 🚨

Please do not publicly post sensitive account data, save data, tokens, or exploit details in an issue.

If you find a vulnerability, open a minimal GitHub issue that describes the affected area without exposing private data, or contact the maintainer through the public contact route listed by the `Corbeno/Idleon-Api-Downloader` repository owner.

Include:

1. A short summary of the issue.
2. Steps to reproduce without private save data.
3. The browser and extension version or commit used.
4. Whether the issue affects copied/downloaded data, cached extension storage, or page injection.

## Scope 🔍

This extension runs locally in Chrome and interacts with `legendsofidleon.com`. Issues in Idleon itself, Chrome, Firebase, or third-party spreadsheets are outside this project's direct control.
